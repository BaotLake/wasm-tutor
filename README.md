


## 环境

### Emscripten
使用docker image
> `docker pull emscripten/emsdk`

> `docker run -i -t -v <your_work_dir>:/src emscripten/emsdk`

> `docker run -i -t -v <your_work_dir>:/src --platform linux/amd64 emscripten/emsdk`


### HTTP Server

Run Python Http Server
    
> `python3 -m http.server 3000`
